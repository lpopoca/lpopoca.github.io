Reference Entry Finder

Reference Entry Finder (REF) is a desktop application used to record notes tied to a reference number. REF was created to fill a gap in simple reference notation software. During my time working as a customer service respresentative I often had to leave voicemails to different organization and individuals. I very quickly learned that I would need a way to quickly search up the reason for my calls when a person did call back. Often times it would be reguarding personally identifiable information so I couldn't leave a name behind on the voicemail. So I created a spreadsheet that would hold reference number entries with notes and timestamps. The thought of creating a stand alone application for searching up reference numbers was always running through my mind so REF was forged into existance.

REF is still in development. You can View source code on github to follow its progress.
